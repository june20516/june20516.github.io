reset vs revert

  • reset : 푸시하지 않은 커밋들에만 쓰자. 특정시점의 커밋 상태로 되돌림 git reset --hard "id"
  • revert : 커밋을 취소한 상태의 새 버젼을 만든다. 즉, 로그가 남음 git revert "id"
